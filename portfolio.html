Project 1.1.9: Cake Maker- For this project my partners and I created a program that creates a cake based one how many layers, candles and icing the user wants. There are three options for the cake layers: strawberry, vanilla, and chocolate, and the cake can have as many layers as the user wants. When the user no longer wants to add layers, the program will ask for the radius which determines the size of the cake. Once the cake is built, it will ask how many candles the user would like. Finally, the program will ask how many rings of icing the user wants and the cake is complete!
